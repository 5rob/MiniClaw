# Notification Sound

Place your notification sound file here as `ding.mp3`.

**Requirements:**
- Short duration (0.5-1 second recommended)
- MP3 format
- Soft/gentle tone

**Free sources:**
- https://notificationsounds.com/
- https://freesound.org/
- https://pixabay.com/sound-effects/

**Example sounds to search for:**
- "notification chime"
- "gentle bell"
- "soft ding"
- "marimba note"

If no sound file is present, the bot will skip the notification and go straight to TTS.
