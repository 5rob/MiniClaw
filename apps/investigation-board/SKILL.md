# Investigation Board

## Overview
This is a **standalone React web application**, not a MiniClaw skill.

The investigation board is an interactive detective-style entity relationship visualizer built with React Flow and d3-force physics.

## Setup Instructions
See `README.md` for installation and development instructions.

## Project Type
- **Type**: Standalone React + TypeScript + Vite application
- **Framework**: React 18 + Vite
- **Visualization**: React Flow v12 + d3-force
- **Styling**: Tailwind CSS

## What It Does
- Visualizes entities (people, companies, organizations) and their connections
- Interactive graph with custom nodes and edges
- Floating word clouds using d3-force physics
- Topic-based filtering and search
- Detective noir dark theme
- Includes comprehensive seed data for a fictional corporate investigation

## Running the App
```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.
